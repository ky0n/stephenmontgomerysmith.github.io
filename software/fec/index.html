The "Fast Exact Closure" Package

This software solves the second moment for the fiber orientation tensor as predicted by Jeffery's type equations. The software is described at http://faculty.missouri.edu/~stephen/preprints/FEC_with_Diffusion.html.

The current version of the software is fec-0.03, which may be downloaded here.
